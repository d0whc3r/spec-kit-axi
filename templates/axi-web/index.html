    How axi works

    1. Read the spec, plan, and tasks using the tabs above the document.
    2. Annotate by selecting any text in the document, then writing a note about that exact passage.
    3. Comment on the whole feature by typing in the box at the bottom and pressing Enter. Notes stack up in the panel so you can review them.
    4. Send the whole batch to the agent with the send button. Anything still in the box goes along with it.
    5. The agent edits the files and replies here. The document reloads on its own to show the changes.
